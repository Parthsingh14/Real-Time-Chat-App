# 🧪 WebSocket Practice Project

This project is a **practice implementation of real-time communication** using WebSockets (Socket.IO).

## 🚀 About the Project

The main goal of this project is to understand how real-time data transfer works between client and server.

It demonstrates:

* Establishing a WebSocket connection
* Sending and receiving messages in real time
* Basic event-based communication using Socket.IO

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Real-time Communication:** Socket.IO

## 📌 Purpose

This is a **learning-focused project** built to:

* Strengthen understanding of WebSockets
* Explore real-time application behavior
* Practice full-stack integration
